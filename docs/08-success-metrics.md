# Success Metrics

- Reduce failure triage time by 80%
- Classify more than 85% of failures deterministically (without AI)
- Deliver feedback within minutes of pipeline completion

These are product-level targets, not per-milestone acceptance criteria. They exist to keep
implementation decisions honest: e.g. a design that pushes more failures into AI-only
classification (rather than deterministic rules) works against the >85%-deterministic target and
should be justified explicitly if proposed.
